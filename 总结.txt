1.对svg图标制作icon图标 使用阿里巴巴inconfont图标网站
1.1 点击资源管理——我的项目-新建项目
1.2 上传svg图标  添加到项目
1.3 点击font-class 查看在线连接 点击链接 全选把样式复制
1.4 新建icon.less文件 把复制的样式粘贴到文件里
1.5 在index.less 用 @import 'icon.less路径' 导入

2.浏览器图标是在public文件夹的ico文件中

3.引入vant组件

4.REM适配
4.1 安装 npm i amfe-flexible 插件
4.2 然后再main.js导入模块
4.3 安装 npm i postcss-pxtorem -D 将px转为rem
4.4 在项目根目录新建postcss.config.js 写上
module.exports = {
  plugins: {
    'postcss-pxtorem': {
      rootValue: 37.5,
      propList: ['*'],
    },
  },
};
4.5 重启服务
4.6 注意：行内样式的px单位转换不了rem
